# Foundation Watering

An app for calculating how much time to water a foundation in The Colony, Texas

## Setup

1. Create virtual environment
	```shell
	python3 -m venv venv
	```
2. Source activation script
	```shell
	source venv/bin/activate
	```
3. Set flask source
	```shell
	export FLASK_APP=display/src/main/app.py
	```
## Run

4. Run the front end
	```shell
	flask run
	```
